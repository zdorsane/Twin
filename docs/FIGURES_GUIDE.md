## Regenerating figures

All figures can be regenerated without running the full training pipeline:

```bash
# Tanimoto analysis (needs Dataset/graphga_tanimoto_vs_ccle.csv)
python scripts/tanimoto_analysis.py

# Molecular validation (needs Dataset/graphga_tanimoto_vs_ccle.csv + brics_dqn_results.csv)
python scripts/molecular_validation.py

# Notebook figures (needs logs/ CSV files)
cd notebooks && jupyter nbconvert --to notebook --execute evaluation.ipynb
```

> **Note:** `logs/` is gitignored. Training CSVs are embedded as fallback values
> in the notebook cells, so the notebook executes correctly even without the
> log files present.

## Colour conventions across figures

| Colour | Meaning |
|--------|---------|
| Blue (`#2196F3`) | Training metrics |
| Red (`#F44336`) | Validation metrics / Lipinski violation |
| Green (`#4CAF50`) | Pearson r / MedChem OK |
| Orange (`#FF9800`) | Bi-Int model |
| Gold | QED ≥ 0.9 (excellent) |
| `#EF9A9A` (light red) | Best classical baseline (XGBoost) |
